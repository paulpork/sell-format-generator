# Item Describer Site

This project is a simple web application that generates formatted paragraphs based on user input for item details. Users can input the item name, buy price, sell price, and whether the item is new or not. The application will then display a formatted description of the item.

## Project Structure

- `index.html`: The main HTML document containing the structure of the webpage.
- `css/styles.css`: Styles for the website, defining layout, colors, and fonts.
- `js/app.js`: JavaScript code that handles user input and generates the formatted paragraph.
- `data/samples.json`: Sample data in JSON format for testing or demonstration purposes.
- `README.md`: Documentation for the project.
- `404.html`: Custom 404 error page for non-existent routes.

## Getting Started

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd item-describer-site
   ```

3. Open `index.html` in your web browser to view the application.

## Usage

1. Enter the item name in the designated input field.
2. Input the buy price and sell price.
3. Select whether the item is new or used.
4. Click the "Generate Description" button to see the formatted paragraph displayed on the page.
